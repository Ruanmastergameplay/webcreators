# Webcreator

Webapp Next.js com Tailwind CSS para portfolio/landing page de criacao de sites, automacoes simples e presenca digital para pequenos negocios.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra:

```text
http://127.0.0.1:3000
```

## Build

```bash
npm run build
```

## Deploy

```bash
vercel --prod
```
